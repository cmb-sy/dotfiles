# Claude 用プロンプトファイル

`claude` コマンドのエイリアス（clr / cld / clc）で、ここにあるテキストが追加のシステムプロンプトとして渡されます。

## コマンド一覧

| 短いコマンド | 別名 | プロンプト | 引数なしのときの自動クエリ |
|--------------|------|------------|----------------------------|
| **clr** | claude-review | review.txt | カレントのコードをレビュー |
| **cld** | claude-doc | doc.txt | カレントのドキュメント作成・整備 |
| **clc** | claude-commit | commit.txt | ステージ済みを確認し、commit メッセージ案を出して commit & push |

- `clc` だけ実行 → 変更確認 → メッセージ案 → commit & push の流れで対話開始
- `clc "feat: ログイン処理を追加"` のように指示を渡すことも可能

## 追加の仕方

1. このフォルダに `名前.txt` を追加する
2. `.aliases.sh` に同じ名前の関数（とデフォルトクエリ）を追加する
